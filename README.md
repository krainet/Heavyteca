# [HeavyTeca](http://github.com/krainet/Heavyteca) 

## Quick Start

Install Node.js and then:

```sh
$ git clone git://github.com/krainet/Heavyteca
$ cd Heavyteca
$ sudo npm -g install grunt-cli karma bower
$ sudo npm install
$ bower install
$ grunt watch
```

Finally, open `http://localhost:9000/` in your browser.
